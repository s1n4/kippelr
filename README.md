# kippelr [![Build Status](https://travis-ci.org/s1n4/kippelr.png?branch=master)](https://travis-ci.org/s1n4/kippelr)

Erlang library for the Kippt API

## Build

```
./rebar get-deps compile
```

## QuickStart

```
erl -pa ebin deps/*/ebin -s kippelr
```
